# 🚀 Asterisk: AI 기반 게임 소개 및 유저 분석 플랫폼

> 1인 개발 게임 'Asterisk'의 홍보, 가이드, 그리고 유저 여론 분석을 위한 **AI 통합 웹 플랫폼**입니다.

<br/>

## 1. 프로젝트 개요 및 기획 의도

본 프로젝트는 단순한 게임 홍보 사이트를 넘어, **생성형 AI(LLM)** 기술을 접목하여 1인 개발자가 겪는 리소스 부족 문제를 해결하는 것을 목표로 합니다.

* **문제점:** 혼자서 게임 개발과 유저 대응(CS)을 병행하기 어렵고, 수집된 수많은 피드백을 일일이 읽고 분석하여 개선점을 도출할 시간이 부족함.
* **해결책:**
    1.  **AI 챗봇 가이드:** 유저의 게임 관련 질문에 24시간 자동 응답하여 CS 부담 감소.
    2.  **커뮤니티 피드백 종합 분석:** 지금까지 수집된 모든 유저들의 피드백을 AI가 한 번에 분석하여, **'한 줄 요약'**과 **'핵심 키워드'**로 정리해 개발자와 유저에게 제공.

<br/>

## 2. 기술 스택 및 선정 이유

본 프로젝트는 **비용 효율성**과 **개발 생산성**을 최우선으로 고려하여 다음과 같은 아키텍처를 구성하였습니다.

| 구분 | 기술 스택 | 선정 이유 및 역할 |
| :--- | :--- | :--- |
| **Frontend** | **HTML5, CSS3, JavaScript** | 별도의 빌드 과정 없이 가볍고 빠르게 배포 가능하며, `Axios`를 통해 비동기 API 통신 구현. |
| **UI Framework** | **Bootstrap 5** | 반응형 웹 디자인을 신속하게 구축하고, 세련된 리포트 카드 및 채팅 UI 제공. |
| **Backend (AI)** | **Google Colab + Python** | 고성능 GPU가 필요한 LLM을 로컬 장비 없이 무료로 구동하기 위한 클라우드 런타임 환경. |
| **API Server** | **FastAPI + uvicorn** | Python 기반의 비동기 웹 프레임워크로, LLM 추론 결과를 JSON 형태로 프론트엔드에 빠르게 전달. |
| **Tunneling** | **ngrok** | 로컬(Colab)에서 실행 중인 FastAPI 서버를 공인 IP 없이 외부(웹사이트)에서 접속할 수 있도록 터널링 제공. |
| **LLM Engine** | **LangChain + Polyglot-ko** | 한국어 성능이 우수한 오픈소스 모델(`Polyglot-ko-1.3b`)을 `LangChain` 파이프라인으로 구축하여 체계적인 답변 및 요약 생성. |
| **Database** | **Google Sheets + GAS** | 별도의 서버 DB 구축 비용 없이, `Google Apps Script`를 통해 데이터를 무료로 저장하고 관리(Excel 호환). |
|**Game Engine** | **Unity Engine** | 2D 게임 개발에 가장 적합한 유니티 엔진을 사용해 게임 개발 진행 |

<br/>

## 3. 시스템 동작 흐름

### 🔄 Flow 1: AI 챗봇 가이드
1.  **User:** 웹사이트 우측 하단 챗봇 버튼 클릭 → 질문 입력 (예: "이 게임 장르가 뭐야?")
2.  **Frontend:** `Axios`를 통해 Colab 서버의 `/chat` 엔드포인트로 질문 전송.
3.  **Backend (Colab):**
    * `LangChain`이 미리 주입된 **게임 기획 데이터**를 프롬프트에 포함.
    * `LLM`이 문맥을 파악하여 게임 설정에 맞는 답변 생성.
4.  **User:** 화면에 AI의 답변이 말풍선 형태로 표시됨.

### 🔄 Flow 2: 커뮤니티 피드백 종합 분석
1.  **User:** 웹사이트 접속 또는 '새로고침' 버튼 클릭.
2.  **Frontend:** `Google Apps Script`에 요청하여 **지금까지 저장된 모든 피드백 데이터**를 가져옴.
3.  **Frontend → Backend:** 가져온 텍스트 데이터를 묶어서 Colab 서버의 `/summarize` 엔드포인트로 전송.
4.  **Backend (Colab):**
    * LLM이 수십 개의 리뷰를 읽고 전체적인 여론을 분석.
    * **전체 내용을 아우르는 한 줄 요약**과 **가장 많이 언급된 키워드 3개**를 추출.
5.  **Frontend:** 분석 결과를 받아 **'유저 피드백 분석 리포트'** 섹션에 시각화하여 보여줌.

## 4. 핵심 기능 상세

### 🎮 1. Asterisk 데모 버전
* **특징:** 현재 개발 중인 게임의 데모 버전을 플레이할 수 있습니다.
* **구현:** Unity Engine을 사용하여 게임을 제작했습니다.

### 🤖 2. Asterisk AI 가이드
* **특징:** 단순한 규칙 기반 챗봇이 아니라, 자연어 이해가 가능한 LLM 기반 챗봇입니다.
* **구현:** `Few-shot Prompting` 기법을 적용하여, 게임의 세계관, 시스템, 개발 로드맵을 학습시켰습니다. 유저가 "출시일이 언제야?"라고 물으면 "2026년 하반기 예정입니다!"라고 정확히 답변합니다.

### 📢 3. AI 커뮤니티 분석 리포트
* **특징:** 개별 피드백을 하나하나 읽는 대신, AI가 전체 데이터를 보고 현재 게임의 여론을 실시간으로 브리핑해줍니다.
* **구현:**
    * **한 줄 요약:** "유저들은 전반적으로 그래픽에 만족하지만, 초반 난이도에 어려움을 느끼고 있습니다."와 같이 핵심을 찌르는 요약 제공.
    * **키워드 추출:** `#난이도`, `#그래픽`, `#기대됨` 등 현재 가장 핫한 주제를 태그 형태로 시각화.
    * 이를 통해 개발자는 업데이트 방향성을 잡고, 신규 유저는 게임의 분위기를 빠르게 파악할 수 있습니다.

## 5. 실행 및 테스트 방법

### Step 1. AI 백엔드 서버 실행 (Google Colab)
1.  `Asterisk_Backend.ipynb` 파일을 Google Colab에서 엽니다.
2.  상단 메뉴에서 **런타임 > 런타임 유형 변경**을 클릭하고 `T4 GPU`로 설정합니다.
3.  **모두 실행**을 눌러 필요한 라이브러리를 설치하고 서버를 구동합니다.
4.  마지막 셀의 실행 결과에 출력된 **`ngrok public url`**을 복사합니다.

### Step 2. 프론트엔드 연결 설정
1.  프로젝트 폴더의 `index.html` 파일을 엽니다.
2.  `API_BASE_URL` 변수를 찾아, 위에서 복사한 주소로 교체합니다.
3.  파일을 저장합니다.

### Step 3. 웹사이트 실행
1.  VS Code의 **Live Server** 확장 프로그램을 사용하여 `index.html`을 엽니다. (또는 브라우저로 직접 실행)
2.  **챗봇 테스트:** 우측 하단 챗봇 아이콘을 눌러 "게임 장르가 뭐야?"라고 질문해 봅니다.
3.  **리포트 테스트:** 웹사이트 하단의 **'유저 피드백 AI 분석 리포트'** 섹션에서 요약 문구와 키워드가 자동으로 로딩되는지 확인합니다.

<br/>

## 6. 결론 및 기대 효과

이 프로젝트는 **SWAI 기술(LLM, FastAPI, LangChain)**을 실제 비즈니스 시나리오(게임 마케팅 및 운영)에 성공적으로 적용한 사례입니다. 이를 통해 1인 개발자는 **운영 비용을 0원에 가깝게 유지**하면서도, 대형 게임사 못지않은 **지능형 유저 서비스**를 제공하고 **데이터 기반 의사결정**을 내릴 수 있습니다.
